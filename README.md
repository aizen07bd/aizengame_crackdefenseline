# 균열 방어선

스마트폰 세로 화면 기준으로 만든 1차선 자동 디펜스 게임 프로토타입입니다.

몬스터는 위에서 아래로 몰려오고, 아군은 바리케이트 뒤에서 자동으로 공격합니다. 플레이어는 전투 중 보상 카드와 캐릭터 선택을 통해 성장 방향을 고릅니다.

## 실행 방법

정적 웹 게임이라 별도 빌드 없이 웹 서버로 열면 됩니다.

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000`으로 접속하면 실행됩니다.

## 현재 구성

- Phaser 기반 2D 디펜스 게임
- 판타지/현대 맵 10종
- 캐릭터 6종
- 일반 공격, 스킬 1/2, 보상 카드
- 컷씬, 인트로, 에필로그, 엔딩 크레딧
- MP3 배경음 및 효과음
